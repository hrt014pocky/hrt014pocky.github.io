东京5天4晚旅游攻略

想去的地方

  • 景点：浅草寺、晴空塔、东京铁塔、明治神宫、根津美术馆、SHIBUYA SKY
  • 游玩：teamLab Borderless、歌舞伎町、Sky Bus Tokyo Marunouchi、浅草浴衣体验
  • 购物：东京THE-FOUR-EYED买手店（新宿区）、𝐂𝐮𝐫𝐢𝐨𝐬 𝐕𝐈𝐍𝐓𝐀𝐆𝐄 𝐂𝐋𝐎𝐓𝐇𝐄𝐒
  • 美食：烤肉 Ushigoro、寿司 Omakase、鳗鱼饭、关东煮
  • 地点：银座奥野大楼、角川、chiikawaちいかわ除草证大头贴、杀死比尔的居酒屋

舍弃：角川（远，性价比低）、超稀有桃子苹果（关门了）、中目黑三明治店（永久停业）、chiikawaちいかわ除草证大头贴（目前没有ちいかわ款式了，可以选择其他）

酒店

Mitsui Garden Hotel Roppongi Tokyo Premier（三井花园酒店六本木东京普米尔）

行程

简要行程

  • 2026-05-02（周六） 到达东京：成田入境，进城入住，晚上直接去 Kill Bill 居酒屋。
  • 2026-05-03（周日） 东京东线 + 银座：浅草浴衣体验、浅草寺、晴空塔、Sky Bus Tokyo Marunouchi、银座奥野大楼，晚上吃 Ushigoro 银座烤肉。
  • 2026-05-04（周一） 东京中线 + 六本木晚餐：teamLab Borderless、麻布台、东京铁塔，傍晚回酒店附近，晚上吃 SUSHIROKU 六本木寿司 Omakase。
  • 2026-05-05（周二） 东京西线 + 夜生活：明治神宫、原宿 / 表参道、Curios Vintage Clothes、涩谷、SHIBUYA SKY、THE FOUR-EYED、歌舞伎町。
  • 2026-05-06（周三） 返程日：酒店周边慢慢收尾，提前去成田机场。

每日详细行程

已定酒店

  • Mitsui Garden Hotel Roppongi Tokyo Premier（三井花园酒店六本木东京普米尔）
  • 地址：3-15-17 Roppongi, Minato-ku, Tokyo 106-0032

航班信息（已确认）

  • 去程：2026-05-02（周六） CA919，14:35 上海浦东 PVG T2 起飞，18:45 东京成田 NRT T1 到达。
  • 返程：2026-05-06（周三） CA930，15:30 东京成田 NRT T1 起飞，17:50 上海浦东 PVG T2 到达。

5.2（周六，到达日）

  • 14:35 从上海浦东起飞，18:45 抵达成田机场 T1。
  • 18:45-20:00 入境、取行李、买交通卡或处理网络，给第一晚留一点缓冲更稳。
  • 20:00-21:40 进城去酒店。
    推荐路线：成田机场 T1 -> Narita Express 到 Tokyo Station -> 东京地铁丸之内线到 Kasumigaseki -> 东京地铁日比谷线到 Roppongi -> 5 号出口步行约 5 分钟到酒店
  • 21:40-22:10 办理入住、放行李、简单洗漱整理。
  • 22:20-23:40 晚餐去 Gonpachi Nishi-Azabu（Kill Bill 居酒屋），第一晚这样排最省脑子，也最贴近酒店。
  • 23:50 左右回酒店休息。
  • 如果航班或入境延误：这晚就直接缩成 入住 + 便利店补给，Gonpachi 改成 5.5 晚上的机动备选，或者这次直接舍弃。

5.3（周日，浅草 + 晴空塔 + Sky Bus + 银座烤肉）

  • 10:00 从酒店出发，坐地铁去 浅草。
  • 10:40-11:40 浅草浴衣店选款、换装、发型整理。
  • 11:40-13:00 穿浴衣逛 浅草寺 + 雷门 + 仲见世商店街。
  • 13:00-14:00 午餐吃 鳗鱼饭。
  • 14:00-14:20 返回浴衣店归还浴衣、取随身物品。
  • 14:20-14:40 前往 东京晴空塔。
  • 14:40-15:50 东京晴空塔 / Solamachi。
  • 15:50-16:20 从晴空塔移动去 Marunouchi Mitsubishi Building 一带。
  • 16:30-17:20 坐 Sky Bus Tokyo Marunouchi，营业时间按 09:30-18:30 记录；实际出发时间按当日班次微调。
  • 17:20-17:40 从丸之内移动去 银座。
  • 17:40-18:20 快速逛 银座奥野大楼，顺便看看银座背街的老建筑气质。
  • 18:30-20:30 晚餐吃 Yakiniku Ushigoro Ginza Namiki St.（焼肉うしごろ 銀座並木通り店），就在银座，接在奥野大楼后面最顺。
  • 20:30-21:30 饭后在银座 / 有乐町轻松散步，或者直接回六本木酒店休息。
  • 22:00 左右回酒店。

5.4（周一，teamLab + 东京铁塔 + 六本木寿司）

  • 10:00 从酒店出发，步行或短程地铁去 teamLab Borderless。
  • 10:30-13:00 teamLab Borderless，建议直接订 10:30 或 11:00 入场，和你晚出门的节奏最匹配。
  • 13:00-14:00 在 麻布台之丘 吃午饭。
  • 14:00-14:40 麻布台之丘 轻松逛一段，或者找咖啡店坐一会儿。
  • 14:40-15:10 前往 增上寺 / 东京铁塔。
  • 15:10-16:40 增上寺 + 东京铁塔，这条线和六本木酒店区非常顺。
  • 16:40-17:20 回 六本木 / 酒店，休息一下、换衣服或放东西。
  • 17:20-18:20 酒店附近机动：六本木Hills、东京中城 或找咖啡店坐一会儿。
  • 18:30-20:30 晚餐吃 SUSHIROKU Roppongi Tokyo，作为这次的寿司 Omakase。
  • 20:30-21:30 饭后在六本木周边散步，状态好再补一个小酒吧；不想折腾就直接回酒店。

5.5（周二，明治神宫 + 原宿表参道 + 涩谷 + 歌舞伎町）

  • 10:00 从酒店出发，前往 明治神宫。
  • 10:30-11:40 逛 明治神宫，先把这天最安静的一段留给神社和树林。
  • 11:45-12:30 走到 原宿 / 表参道，顺路找午餐。
  • 12:30-14:00 原宿 / 表参道 逛街拍照。
  • 14:00-15:00 去 Curios Vintage Clothes，这类古着店建议临近出发前再确认营业日。
  • 15:00-16:00 机动安排咖啡店休息，或者继续补逛表参道。
  • 16:00-16:30 前往 涩谷。
  • 16:30-18:00 SHIBUYA SKY，建议提前在官网买日落前后的时段；如果没抢到票，就改成涩谷自由逛。
  • 18:00-18:40 从涩谷移动去 新宿，晚高峰给一点缓冲。
  • 18:40-19:30 去 THE FOUR-EYED（新宿区），再开始晚上的新宿动线会更稳。
  • 19:30-22:00 歌舞伎町 夜逛 + 晚餐 / 小酒吧；想保留 关东煮 的话，放在这一晚做备选。
  • 22:30 左右回酒店。

5.6（周三，返程日）

  • 08:30-09:30 起床、收拾行李、退房前做最后整理。
  • 09:30-10:30 在酒店附近吃早餐，或者去 东京中城 / 六本木 最后慢慢走一圈。
  • 10:30-10:50 回酒店拿行李、正式退房。
  • 11:00-12:30 从六本木出发去 成田机场 T1；返程日这里不按 10:00 出门规则走，要优先给国际航班留时间。
  • 12:30-14:30 机场值机、安检、简单午餐、免税店收尾。
  • 15:30 CA930 起飞，17:50 抵达上海浦东 T2。
