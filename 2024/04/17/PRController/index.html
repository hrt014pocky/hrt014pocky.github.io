PR Controller 比例谐振控制器离散化，双积分器形式

发表于 2024-04-17 更新于 2024-06-06

在直流控制算法里面，电流内环的输入偏差信号为基波 50Hz 而且谐波成分很多的交流信号。所以，传统的 PI 控制器难以完成无稳态误差控制这一任务，也难以实现谐波抑制，不符合高精度控制的标准。所以引入PR控制，实现了对并网电流的零稳态误差跟踪。

传递函数

PI控制器的传递函数：

G(s)=kp+kisG\left( s \right) =k_p+ \frac{k_i}{ s} G(s)=kp​+ski​​

PR控制器的传递函数：

G(s)=kp+ss2+ω2G\left( s \right) =k_p+ \frac{s}{ { {s^2} + {\omega ^2}}} G(s)=kp​+s2+ω2s​

准PR控制器的传递函数：

G(s)=kp+krωcss2+ωcs+ω02G\left( s \right) =k_p+ \frac{ { {k_r}{\omega _c}s}}{ { {s^2} + {\omega _c}s + {\omega _0}^2}} G(s)=kp​+s2+ωc​s+ω0​2kr​ωc​s​

kp 为比例系数; kr 为谐振系数; ω0 = 2πf 为基波角频率; ωc为截止频率。

绘制PI控制器和准PR控制器的波特图：

PR 控制器在选定频率时的增益大约是 40dB。但是 PI 控制器在这一频率时幅值增益非常低。幅值增益可以看出控制器对控制系统输入偏差的放大幅值，增益越大那么误差信号衰减速度越快，相应的谐波抑制效果越好。
