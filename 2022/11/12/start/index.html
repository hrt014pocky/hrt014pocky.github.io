个人博客搭建方法 使用Hexo+Github

发表于 2022-11-12 更新于 2023-10-23 分类于 博客搭建

前言

本文将介绍使用Hexo+Github从0到1搭建自己的个人博客。

Hexo框架搭建

Hexo 是一个基于Nodejs的博客框架，快速、简洁且高效，可以方便的生成静态网页托管在GitHub或者Gitee上面。在安装Hexo之前首先需要确保自己的电脑已经正确安装了Git和Nodejs。

Git安装

Git安装下载地址：https://git-scm.com/downloads

  • windows:Git官网下载并安装。
  • Mac：使用 MacPorts 或者下载安装程序。
  • Linux (Ubuntu, Debian)：sudo apt install git-core
  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

Nodejs安装

官方网站下载安装包进行安装 https://nodejs.org/en/download/

Hexo安装

必备的应用程序安装完成之后，即可以使用npm安装Hexo。

next主题的Github地址： https://github.com/theme-next/hexo-theme-next

使用Github建立博客网站

以上我们在以及建立了一个博客框架，但是只能在本地访问。想要在互联网上查看我们写好的文章，还需要部署到网站上。在里我使用的是Github Pages，它提供了静态网页托管服务，无须自己购买云服务进行搭建，并且是是完全免费的。

Github账号注册与代码仓库创建

  1. 进入Github官网，点击Sign up进行注册。
  2. 进入代码仓库 Your repositories ,点击New创建一个新的仓库。
  3. 注意代码仓库的名称为 “[你的Github用户名].github.io”，点击Create repositories新建代码仓库。
    4.进入创建的代码仓库，点击Create new file新建一个index.html的文件。输入内容
  1. 进入创建的代码仓库，选择Settings,打开Pages选项，点击GitHub Pages
  2. 浏览器打开一个网页，输入网站：https://[你的Github用户名].github.io/ ，如果显示test文字说明创建Github Pages个人主页成功，如果显示找不到网页，说明搭建失败。

部署博客到Github Pages

安装 Hexo 发布插件, 输入下面的命令，安装hexo-deployer-git

打开站点配置文件/_config.yaml，修改deploy属性

发布

打开浏览器，输入网址https://[你的Github用户名].github.io
